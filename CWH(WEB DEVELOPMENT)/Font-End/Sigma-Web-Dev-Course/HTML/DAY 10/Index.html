HTML Entities Demonstration

Basic HTML Entities

To display special characters, use HTML entities:

  • Ampersand: &
  • Less Than: <
  • Greater Than: >
  • Non-breaking Space:   (space)
  • Double Quotation Mark: "
  • Single Quotation Mark: '

Extended HTML Entities

Some common symbols include:

  • Copyright Symbol: © 2024 My Website
  • Registered Trademark Symbol: ® MyCompany
  • Euro Sign: 100 €
  • Degree Symbol: 90° Celsius
  • Bullet Point: • Item 1 • Item 2
  • Section Symbol: Section §1
